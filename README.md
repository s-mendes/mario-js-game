# Mario Game JS

## Tecnologias Utilizadas

- HTML
- CSS
- Javascript

## Objetivo
Criar um mini-game que apertando qualquer tecla, o personagem irá pular os obstáculos e contabilizar os pontos.

- Utilizar localstorage e registrar recorde do jogador.
